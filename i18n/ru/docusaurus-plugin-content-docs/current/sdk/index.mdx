# Работа с SDK

## Интро

На данный момент существует только серверный SDK для Node.js, однако вы можете использовать примеры
из [Тестового приложения](testapp.md), чтобы реализовать интеграцию в JavaScript на клиентской стороне. Релиз SDK для Unity
запланирован на май 2025.

Также доступны API References для [TeleStore](../api-telestore) и [TrexWallet](../api-trexwallet), платежной системы TeleStore, для расширенных возможностей работы с кошельком.
